## 📦 项目简介

**AntiAPP 数据分析工具完整版** - 基于 Flask 的 Web 数据分析应用

### 核心功能
- ✅ 多因子方差分析（ANOVA）
- ✅ LSD 多重比较
- ✅ PCA 主成分分析（含碎石图、双标图）
- ✅ 聚类分析（K-Means、层次聚类）
- ✅ Excel 数据导入导出
- ✅ 侧边栏布局仪表盘

### 技术栈
- Python 3.10+
- Flask 2.3.0
- Pandas, NumPy, SciPy
- statsmodels, scikit-learn, matplotlib

## 🚀 快速开始（新电脑部署）

### 方法 1：从 GitHub 克隆（推荐）

```bash
# 1. 克隆仓库
git clone https://github.com/Ayanya-0628/datatool-advanced.git
cd datatool-advanced

# 2. 创建虚拟环境
python -m venv .venv

# Windows 激活虚拟环境
.venv\Scripts\activate

# Linux/Mac 激活虚拟环境
source .venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行应用
python app.py
```

访问：http://localhost:7860

## 📊 使用指南

### 1. 上传数据
- 支持格式：CSV、Excel (.xlsx, .xls)
- 自动识别因子列（分类变量）和指标列（数值变量）

### 2. 方差分析
- 选择因子列和指标列
- 自动执行方差分析和 LSD 多重比较
- 生成显著性字母标记
- 导出 Excel 结果（6个工作表）

### 3. PCA 主成分分析
- 选择多个指标进行主成分分析
- 自动生成：
  - 碎石图（Scree Plot）
  - 2D/3D 双标图（Biplot）
  - 载荷矩阵
  - 方差解释率

### 4. 聚类分析
- K-Means 聚类
- 层次聚类（树状图）
- 自动生成肘部法则图
- 可视化聚类结果

## 📝 开发指南

### CLAUDE.md 文档
项目包含详细的开发指导文档 `CLAUDE.md`，内容包括：
- 常用命令
- 架构说明
- API 端点
- 统计方法
- 版本记录规范

### 关键设计模式
1. **数据存储：** 内存字典，UUID 索引，1 小时自动过期
2. **行顺序保持：** `groupby(sort=False)` + `pd.Categorical`
3. **列顺序规范：** 因子列在前，指标列在后
4. **可选模块：** 使用 `HAS_*` 标志优雅降级

## 🆘 常见问题

### Q1: 导入模块失败？
```bash
# 确保激活虚拟环境
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac

# 重新安装依赖
pip install -r requirements.txt
```

### Q2: 端口被占用？
```bash
# 修改端口（在 app.py 最后一行）
app.run(host='0.0.0.0', port=5000)  # 改为其他端口
```

### Q3: 上传文件大小限制？
默认 50MB，可在 `app.py` 中修改：
```python
app.config['MAX_CONTENT_LENGTH'] = 50 * 1024 * 1024  # 50MB
```
