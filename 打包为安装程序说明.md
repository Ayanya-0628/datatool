# 将本应用打包为「需要安装的版本」

使用 **PyInstaller** 生成可执行程序，再用 **Inno Setup Compiler** 打成安装包（.exe 安装程序）。

---

## 前提

- 本机已安装 **Python 3.10+** 且已执行过 `pip install -r requirements.txt`
- 已安装 **PyInstaller**：`pip install pyinstaller`
- 已安装 **Inno Setup Compiler**（你已有）

---

## 步骤一：用 PyInstaller 生成可执行程序

在项目根目录（与 `数据分析工具.spec` 同级）打开命令行，执行：

```bash
pyinstaller 数据分析工具.spec --clean --noconfirm
```

- 首次或依赖变更后可能较慢（几分钟到十几分钟）。
- 完成后会在 **`dist\DataAnalysisTool`** 下得到 `DataAnalysisTool.exe` 及依赖文件（整目录即“绿色版”）。

可先双击 `dist\DataAnalysisTool\DataAnalysisTool.exe` 测试：**直接打开系统默认浏览器**，无任何小窗口；关闭浏览器标签页会尝试自动结束服务。

---

## 步骤二：用 Inno Setup 打成安装包

1. 打开 **Inno Setup Compiler**
2. 菜单：**文件 → 打开**，选择项目里的 **`installer_setup.iss`**
3. 菜单：**构建 → 编译**（或按 Ctrl+F9）
4. 编译完成后，在项目下的 **`Output`** 文件夹中会生成 **`数据分析工具_Setup.exe`**

该文件即为可分发的安装程序：用户双击后选择安装路径、是否创建桌面快捷方式等。安装完成后启动「数据分析工具」会**直接打开系统默认浏览器**，无小窗口；关闭浏览器标签页即结束服务。网页内点击「选择目录」时会弹出系统文件夹选择框。

---

## 安装包行为说明

- **安装位置**：默认 `C:\Program Files\数据分析工具`（或 64 位系统的 Program Files）
- **开始菜单**：会创建「数据分析工具」文件夹，内含启动项和「卸载 数据分析工具」
- **桌面快捷方式**：安装时可勾选「创建桌面快捷方式」
- **卸载**：通过「设置 → 应用」或开始菜单中的「卸载 数据分析工具」卸载

---

## 若 PyInstaller 报错

- **缺少模块**：在 `数据分析工具.spec` 的 `get_hiddenimports()` 里为该模块增加一项（如 `'statsmodels.xxx'`），再重新执行步骤一。
- **缺少文件**：若提示缺少 `templates`/`static`/`modules` 下某文件，检查 `datas` 是否包含对应目录（当前 spec 已包含 `templates`、`static`、`modules`）。
- **launcher 无法启动**：在项目根目录用 `python launcher.py` 先确认能正常运行，再打包。
