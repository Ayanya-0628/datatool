# 数据分析工具

多因子方差分析、LSD多重比较、三线表生成的在线工具。

## 功能特性

- 📂 支持 Excel/CSV 文件上传
- 📊 多因子方差分析（含交互效应）
- 🔤 LSD 多重比较（自动字母标记）
- 📋 三线表自动生成
- 📥 Excel 结果导出
- 🌐 响应式设计，支持手机访问

## 快速开始

```bash
# 克隆仓库
git clone <your-repo-url>
cd AntiAPP

# 安装依赖
pip install -r requirements.txt

# 运行
python app.py
```

访问 http://localhost:5000
