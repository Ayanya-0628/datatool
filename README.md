# 📊 数据分析工具 (Python Web 版)

这是一个基于 Web 的简易数据分析工具，旨在为非编程背景的研究人员提供便捷的统计分析服务。

## 🌟 主要功能
*   **多因子方差分析 (ANOVA)**：支持交互作用分析。
*   **LSD 多重比较**：自动生成显著性标记（a, b, c...）。
*   **三线表生成**：一键生成符合学术发表要求的标准表格。
*   **Excel 导出**：结果可直接下载为 Excel 文件。

## 🚀 部署说明
本项目已适配 ModelScope docker 创空间部署。
- **端口**: 7860
- **环境**: Python 3.10 + Flask + Pandas
